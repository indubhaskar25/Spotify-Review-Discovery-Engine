## Quick Reference

**Total estimated duration:** 4–6 weeks (solo developer) or 2–3 weeks (2-person team)

**Phases at a glance**

| Phase | Focus | Duration |
|-------|-------|----------|
| 0 | Foundation & scaffolding | 2–3 days |
| 1 | Data ingestion & cleaning | 4–5 days |
| 2 | Embeddings & vector store | 3–4 days |
| 3 | Retrieval & RAG layer | 3–4 days |
| 4 | AI analysis & insight generation | 4–5 days |
| 5 | Streamlit dashboard | 5–6 days |
| 6 | Export, polish & deployment | 3–4 days |
